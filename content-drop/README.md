# 素材投放区

把待整理的班级活动素材放在这个目录下。

当前约定：

- 你可以直接新建 `test/` 文件夹，把最近一次微信公众号推文的文案和图片放进去。
- 文案格式可以是 `.md`、`.txt`、`.docx`，都可以。
- 图片请尽量保留原图，不需要先压缩。
- 如果有视频链接，可以放一个 `videos.md` 或 `links.txt`，写明标题和链接。

推荐结构：

```text
content-drop/
  test/
    post.md
    images/
    videos.md
    links.md
```

如果你手头素材比较乱，也没关系。只要大致放进 `test/` 里，我会负责：

1. 分析原始文案和图片。
2. 清洗并拆分成活动标题、日期、地点、摘要、正文、图集、视频、延伸链接。
3. 归档成站点可用的数据结构。
4. 接到前端页面里，用于优化首页、活动详情页和相关信息模块。

后续处理原则：

- 优先保留原始信息，不随意删减事实内容。
- 会把公众号口吻整理成更适合站点阅读的档案馆表达。
- 会挑选适合作为封面、详情首图、相册图的图片。
- 会保留后续继续替换 UI 的接口，不把内容写死在页面结构里。
