# Design Context

## Target Audience

The public site serves prospective EMBA students, alumni, school stakeholders, invited guests, and class members revisiting shared activities.

## Use Cases

- Browse the class introduction and public activity archive.
- Read activity recaps with photos and videos.
- Explore class people, guests, annual memories, galleries, and featured videos.
- Leave an email or lightweight contact message for follow-up.
- Let operators publish polished activity records through a visual backend.

## Brand Personality / Tone

Refined, archival, academic, warm, and restrained. The site should feel like a living class yearbook and public memory room, not a commodity SaaS dashboard or hard-sell landing page.

## UI Iteration Principle

The public site must keep content data, view models, components, and design tokens separated so the visual direction can be repeatedly improved without changing CMS data models.
